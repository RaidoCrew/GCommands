# Introduction

If you came here you probably want to learn how to work with the gcommands framework. GComamnds allows you to easily create commands in the discord.js (opens new window) library. For example, this guide will teach you:

  • How to create and run a bot.
  • How to create commands correctly.
  • Easily create slash commands.

This guide will also cover subjects like common errors and how to solve them, keeping your code clean, setting up a proper development environment, etc. If you have a problem, just connect to a discord (opens new window).

# Before you begin...

First, we recommend that you learn at least some basics of javascript. You can start without, but then you can get stuck on many simple problems that you would otherwise solve right away.

  • Modern JavaScript tutorial (opens new window)
  • MDN's JavaScript guide and Documentation (opens new window)
  • CodeCademy JavaScript course (opens new window)
  • W3School JavaScript (opens new window)
  • W3School NodeJS (opens new window)
  • And honestly, Stack Overflow (opens new window) will be solving many problems for you
